# Система управления банковскими картами

Для запуска приложения следует зайти в корень проекта и вызвать команду: 
```
docker compose up -d
```
По умолчанию, будет создан аккаунт с правами админа с
логином `admin` и паролем `password`.
<br>Для изменения данного поведения следует изменить переменные среды `ADMIN_USERNAME` и `ADMIN_PASSWORD` в docker-compose.yaml

---
По умолчанию, приложение будет запущено на порту ```8083```
<br>А swagger-ui будет доступен по <a href="http://localhost:8083/api/v1/swagger-ui.html">данной ссылке</a>
